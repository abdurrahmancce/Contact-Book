# Contact Book UI

A modern and responsive Contact Book web application built using **HTML, CSS, and JavaScript**.  
This project allows users to manage contacts with full CRUD functionality including adding, viewing, searching, updating, and deleting contacts. The application features a clean dark-themed UI, local storage support, real-time search, form validation, toast notifications, and a fully interactive user experience.

---

# Features

- Add New Contacts
- View Contact Details
- Update Existing Contacts
- Delete Contacts
- Real-Time Search Filtering
- Local Storage Data Persistence
- Responsive Modern UI

# Application Overview

The Contact Book application provides a modern dashboard-style interface for managing personal contacts efficiently.

Users can:

- Store contact information
- Search contacts instantly
- Edit contact details
- Delete unwanted contacts
- Persist data using browser local storage

---

# Contact Information Structure

Each contact contains:

| Field | Description |
|---|---|
| Full Name | Contact person's name |
| Phone Number | Unique phone number |
| Email Address | Optional email |
| Address | Optional address |

---

# CRUD Operations

| Operation | Description |
|---|---|
| Create | Add a new contact |
| Read | View contact information |
| Update | Modify contact details |
| Delete | Remove contact permanently |
