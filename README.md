# Contact Book CLI

A modern and beginner-friendly **Contact Book Application** built using **Python**.  
This project allows users to manage contacts directly from the command line using full **CRUD operations**.

The application is simple, clean, and easy to expand for future improvements.

---

# Features

- Add New Contact
- View All Contacts
- Search Contact
- Update Existing Contact
- Delete Contact
- Store Contacts Permanently using JSON
- Prevent Duplicate Phone Numbers
- Clean Terminal Interface
- Input Validation
- Beginner-Friendly Code Structure

---

# Technologies Used

- Python 3
- JSON File Handling
- CLI (Command Line Interface)

# Contact Information Structure

Each contact contains:

| Field | Description |
|---|---|
| Full Name | User's complete name |
| Phone Number | Unique phone number |
| Email Address | Contact email |
| Address | User location/address |

---

# CRUD Operations

| Operation | Description |
|---|---|
| Create | Add a new contact |
| Read | View all contacts |
| Update | Edit contact details |
| Delete | Remove a contact |
